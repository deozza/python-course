# Installation sur MacOS

**Prérequis** :

- avoir accès au terminal bash
- avoir installé [brew](https://brew.sh/fr/)

## Étape 1 : installer python

```bash
brew install python
```

## Étape 2 : vérifier l'installation de python

```bash
python --version
```

Il devrait s'afficher dans le terminal le numéro de version de python d'installé.
