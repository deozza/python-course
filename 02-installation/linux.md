# Installation sur Linux

**Prérequis** : avoir accès au terminal bash

## Étape 1 : mettre à jour les dépendances

```bash
sudo apt-get update -y && apt-get dist-upgrade
```

## Étape 2 : installer les paquets python

```bash
sudo apt-get install python3```

## Étape 3 : vérifier l'installation de python

```bash
python --version
```

La version de python que vous venez d'installer devrait s'afficher dans le terminal.
