# Installation sur windows

## Étape 1 : télécharger l'installeur

Depuis le [site](https://www.python.org/downloads/windows/) de python, choisir l'installeur de la version la plus récente pour windows 64-bit

## Étape 2 - exécuter l'installeur

Exécuter l'installeur.

**Laisser l'installeur ajouter `python.exe` au `PATH`**

## Étape 3 : vérifier l'installation de python

Dans un terminal, lancer :

```bash
python --version
```

La version de python que vous venez d'installer devrait s'afficher dans le terminal.
