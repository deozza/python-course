## Exemple de script

```python
print('Hello world')
```

**Analyse ligne par ligne :**

1. on exécute une instruction pour affiher un message
  - utilisation de la fonction print de python pour afficher un message
  - le message est contenu dans des `'` (*guillements simples* ou *single quotes*)
  - la fin de l'instruction n'a pas de `;` (*point-virgule* ou *semi-column*)

Comme on veut afficher un message de texte, on utilise des `''`. Si on avait voulu afficher un nombre, on aurait pu écrire directement :

```python
print(42);
```

Même chose pour des nombres à virgule :


```python
print(3.14);
```

Comme il n'y a pas de séparateur d'instruction (`;` en PHP ou Java), on ne peut pas écrire plusieurs instructions sur une même ligne. Une ligne équivaut à une instruction.
